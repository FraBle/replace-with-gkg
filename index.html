replace-with-gkg¶

Replace values with suggestions from Google Knowledge Graph.

replace-with-gkg [OPTIONS] COMMAND [ARGS]...

Options

-k, --gkg-api-key <gkg_api_key>¶

process-file¶

Replace values with suggestions from GKG for given CSV file.

replace-with-gkg process-file [OPTIONS] COLUMN CSV_FILE

Options

-i, --in-place¶

Replace CSV file values in-place (default: false)

-o, --output-file <output_file>¶

Output CSV file path (ignore when using –in-place; default stem(<CSV_FILE>)_out.csv)

-s, --save-openrefine¶

Save replacements as OpenRefine Operation History file (default: false)

-f, --openrefine-output-file <openrefine_output_file>¶

OpenRefine Operation History file path (default stem(<CSV_FILE>)_openrefine.json)

-c, --save-processed-values¶

Store processed values in file (default: false)

-r, --processed-values-output-file <processed_values_output_file>¶

Processed values file path (default stem(<CSV_FILE>)_processed.json)

-g, --ignore-values-file <ignore_values_file>¶

Load values to be ignored from file

-d, --dry-run¶

Skip replacing and saving CSV file values (default: false)

Arguments

COLUMN¶

Required argument

CSV_FILE¶

Required argument
